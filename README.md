## 📌 Overview

**Shorty** is a URL shortener web application designed with simplicity, speed, and visual appeal in mind. It helps users transform long URLs into compact links that are easier to share, manage, and use across social media, messaging apps, presentations, and marketing campaigns.

Whether you want a cleaner link for a post or a more professional-looking URL for your workflow, Shorty provides a smooth and intuitive interface to get the job done.

---

## ✨ Features

### 🚀 Core Functionality
- **Instant URL shortening** — convert long links into short, readable URLs.
- **Quick sharing** — generate links that are easier to copy and distribute.
- **Simple workflow** — minimal steps from input to result.

### 🎨 User Experience
- **Modern UI** — clean layout with a premium visual style.
- **Responsive design** — works beautifully on desktop, tablet, and mobile.
- **Fast interactions** — optimized for a lightweight and smooth experience.
- **Clear structure** — easy to understand for first-time users.

### 🔐 Practical Benefits
- **Improved readability** — make long URLs look neat and manageable.
- **Better presentation** — ideal for social posts, bios, documents, and demos.
- **Professional appearance** — enhances the look of shared links.

## ⚙️ Getting Started

### Prerequisites
Make sure you have the following installed:
- **Node.js** (recommended: latest LTS)
- **npm** or **yarn**

### Installation

```bash
git clone https://github.com/Gavinduachintha/Shorty.git
cd Shorty
npm install
```

### Run the project

```bash
npm run dev
```

Then open the app in your browser.

> If your development server uses a different port, check the terminal output after running the command.

---

## 🔧 Environment Variables

If your app uses environment variables, create a `.env` file in the project root and add the required keys.

Example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## 🧪 Usage Guide

1. Open the app in your browser.
2. Paste a long URL into the input field.
3. Click the shorten button.
4. Copy the generated short link.
5. Share it anywhere you want.
